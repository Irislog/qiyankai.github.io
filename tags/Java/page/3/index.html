    枚举：能列举出所有情况、使用枚举，枚举类使用enum 修饰

    枚举注意事项：

    • 1、所有的枚举类继承自java.lang.Enum
    • 2、枚举类中的所有类成员均被public static final 每一个成员其实都是一个被枚举实例出来的对象
    public class EnumDemo {
    public static void main(String[] args) {
        Weekday i = Weekday.SUN;
        //name():获取名字
        System.out.println(i);
        //ordinal():代表位置，从0 开始
        System.out.println(i.ordinal());
        //values():获取枚举类中所有对象
        Weekday[] weekday = i.values();
    }
}
//枚举类型
enum Weekday {SUN,MON,TUS,WED,THU,FRI,SAT}

    什么是异常机制

    什么是异常，比如说：
    医院–》 医生–》 身体某些部位异常–》 该部位和正常相比不同
    将受到影响

    异常：

    并不是语法出错，语法错误，无法编译，不会产生字节码文件，不能运行

    为什么学习异常机制？

    • 异常处理是衡量一门语言是否成熟的标准之一
    • java、c++，c# 等都有异常处理
    • 异常处理都可以让程序有更好的容错性，是代码更加健壮
    • if-else 其实是一种异常

      没有异常存在缺点：

    • 使用方法的返回值来表示异常情况有限，无法琼剧所有的情况
    • 异常流程和正常流程混合在一起，增大程序的复杂性，可读性也不好
    • 随着系统规模的不断扩大，程序的可维护性极低

      针对上述的没有异常机制的缺点，我们提出

    1. 把异不同类型的异常情况描述成不同类
    2. 分离异常流程代码和正常代码不一样

      异常转译：

      当位于上层的子系统不需要关心底层的异常细节时，常见的做法就是捕获异常，把它转换为新的不同类型的异常，再抛出新的异常

      异常链：

      把原始的异常包装为新的异常类，从而形成多个异常的有序排列，有助于查找异常的根本原因

    概念：

    内部类会生成字节码文件，此时文件名称冠以外部类的名称$ ：外部类$ 内部类

    注意: 在内部实例中、不允许使用static修饰成员变量，（除final修饰以外）成员方法

    1.局部内部类

    2.匿名内部类

    new 构造器([参数列表])|接口名称{
    类体
    }
    注意: 匿名内部类只能继承一个类或者实现一个接口,此时匿名内部类默认继承

    3.私有内部类

    4.静态内部类

    系统会创建一个继承自Bird类的匿名对象,该对象转型为Bird类型的引用，对于匿名内部类的使用它存在缺陷，仅能被使用一次，创建匿名内部类，什么时候使用，什么时候会创建一个该类的实例，会立即消失

    匿名内部类使用注意事项：

    1. 使用匿名内部类时，我们必须继承一个类或者实现一个接口，但是两者不可兼得，匿名内部类必须继承或者实现一个接口
    2. 匿名内部类是不能定义构造函数的
    3. 匿名内部类中不能存在任何静态变量和静态方法
    4. 匿名内部类为局部内部类，所以局部内部了哦的所有限制对匿名内部类有效
    5. 匿名内部类不能是抽象的，他必须要实现继承或者实现接口的所有抽象方法

    什么是字符串，一串字符

    本质：char[] 数组

    字符串的分类

    1、不可变的字符串：String 类
    2、可变的字符串：StringBuilder 、 StringBuffer 类

    不可变的字符串：

    一个String 对象只能表示一个固定的字符串，
    若内容改变，则对象也改变

    字符串常见问题：

    • 1、单独使用”” 引号创建的字符串都是直接量，编译器就已经确定储存到常量池中
    • 2、使用new String() 创建的对象会存到堆内存中，是运行期才会有
    • 3、使用只包含直接量的字符串链接符，如”aa”+”bb” 创建的也是直接量编译期就能确定，已经确定存储到常量池中
    • 4、使用String 直接量（无final 修饰）的字符串表达式如”aa”+temp 创建的对象试运行期才创建得存储在堆中的0

    String中的常用方法:

    • toUperCase
    • toLowerCase
    • starts
    • toCharArray
    • valueOf
    • subString

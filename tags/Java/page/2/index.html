    2017-03-22
    Java

    枚举：能列举出所有情况、使用枚举，枚举类使用enum 修饰

    枚举注意事项：

    • 1、所有的枚举类继承自java.lang.Enum
    • 2、枚举类中的所有类成员均被public static final 每一个成员其实都是一个被枚举实例出来的对象
    public class EnumDemo {
    public static void main(String[] args) {
        Weekday i = Weekday.SUN;
        //name():获取名字
        System.out.println(i);
        //ordinal():代表位置，从0 开始
        System.out.println(i.ordinal());
        //values():获取枚举类中所有对象
        Weekday[] weekday = i.values();
    }
}
//枚举类型
enum Weekday {SUN,MON,TUS,WED,THU,FRI,SAT}

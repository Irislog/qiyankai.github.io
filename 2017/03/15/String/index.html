    什么是字符串，一串字符

    本质：char[] 数组

    字符串的分类

    1、不可变的字符串：String 类
    2、可变的字符串：StringBuilder 、 StringBuffer 类

    不可变的字符串：

    一个String 对象只能表示一个固定的字符串，
    若内容改变，则对象也改变

    字符串常见问题：

    • 1、单独使用”” 引号创建的字符串都是直接量，编译器就已经确定储存到常量池中
    • 2、使用new String() 创建的对象会存到堆内存中，是运行期才会有
    • 3、使用只包含直接量的字符串链接符，如”aa”+”bb” 创建的也是直接量编译期就能确定，已经确定存储到常量池中
    • 4、使用String 直接量（无final 修饰）的字符串表达式如”aa”+temp 创建的对象试运行期才创建得存储在堆中的0

    String中的常用方法:

    • toUperCase
    • toLowerCase
    • starts
    • toCharArray
    • valueOf
    • subString

    接口

    2017-03-26
    Java

    概念

    表示一种规范，用来定义一组方法，表示某一类事务具备的功能，其实现类必须实现该功能。

    注意

    • 使用interface关键字声明
    • 接口能实例化
    • 一个类可实现（implement）多个接口（中间用逗号隔开），但只能继承一个类
    • 接口中的方法默认使用public abstract修饰
    • 接口中的成员变量public static final修饰
    • 接口中不能实现接口，但允许继承多个接口
    • 接口多继承时，实现最终接口的类必须把所有的父接口的方法实现
    public class InterfaceDemo03 {
    public static void main(String[] args) {
        //提供键盘对象、鼠标对象
        MainBoard.plugin(new Mouse());
        MainBoard.plugin(new KeyBoard());
        MainBoard.doWork();
    }
}
interface IUSB{
    //提供一个交换数据
    void swapData();
}
//鼠标
class Mouse implements IUSB{
    @Override
    public void swapData() {
        System.out.println("鼠标在移动。。。");
    }
}
//硬盘
class KeyBoard implements IUSB{

    @Override
    public void swapData() {
        System.out.println("敲打键盘。。。");
    }
}
//主板
class MainBoard{
    //声明成员变量
    private static IUSB[] usbs = new IUSB[2];
    //声明一个控制当前下标的变量
    private static int index = 0;
    //写一个静态方法，用于将鼠标、键盘插入
    public static void plugin(IUSB iusb){
        if (usbs.length == index){
            System.out.println("USB满了");
            return;
        }
        usbs[index] = iusb;
        index++;
    }
    //写一个方法，用于工作
    public static void doWork(){
        for (IUSB usb : usbs) {
            usb.swapData();
        }
    }
}

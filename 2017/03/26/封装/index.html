    封装

    2017-03-26
    Java

    封装的概念：

    1. 把对象的状态和行为看成一个统一的整体，将而这存放在一个独立的模块中（类中）
    2. 信息隐藏：把不需要让外界知道的信息隐藏起来，尽可能的而隐藏对象的功能实现细节

    问题：

    1. 封装什么类？
    2. 封装什么行为和状态？
    3. 如何隐藏信息？

    举例：void save(String name,int age,int phoneNum,String email)

    但是保存的信息太多了

    解决问题

    联系人封装到一个模块中（类），只需要在方法中传入一个封装好的实体类对象就可以了

    四大访问权限修饰符

    修饰变量方法:

    定义类的时候，使用public，可以不写

    1. private（类访问权限）：修饰字段，方法，构造器，只能在当前类中使用
    2. package（缺省/包访问权限）（不写时）：在同一个包中，才可以访问到
    3. protected（子类访问权限）：即使父类和子类不在同一包中，他也访问
    4. public（公共访问权限）：在任何地方都可以访问

    2、import

    import 导入

    静态导入：告诉编译器去哪一个java.lang 包中的类去找某个静态方法或者字段

    * java.lang
*   语言核心类，系统自动导入，天天用

* java.util
*   工具类，集合框架，时间，日历

* java.net
*   网络编程接口和类以后要写和网络有关的

* java.io
*   流的接口和类，一些要读写文件或者图片格式化相关的类

* java.text
*   格式化相关的类，国际软件

* java.sql
*   接口和类连接数据库的接口和类

* java.awt
*   抽象窗口工具集的接口和类，界面开发

* java.swing
*   图形用户界面相关接口和类（可以跨平台）

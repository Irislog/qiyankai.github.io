    面试题三：创建下面两行代码，分别创建了几个对象？

    1. String name5 = "will";

      创建了1 | 0 个String 对象
      在常量池中已经存在“will” ，则直接引用,如果不存储，先创建一个“will”对象,最多创建一个String 对象，最少不创键。

    2. String name6 = new String ("will");

      创建了1 | 2 个String 对象，new 决定了堆空间的创建对象，必须有一个，若常量池中有“will”，则直接引用，不存在，先创建一个“will”对象，最多有两个对象，至少创建一个
      new 关键字，绝对会在堆空间创建一个内存区域。
